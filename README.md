# InforSystem
demo为前端部分  
使用方式  
`cd demo`  
`npm run serve`  
此方法为开发环境  
要打包部署需调整为`npm run build`  
注意打包后的路径问题  
demorear为后端部分  
使用方式  
使用idea  
在idea中跑该springboot项目
